# Магазин игрушек «ToyStore»

Демонстрационный экзамен 09.02.07, вариант № 4. Информационная система учёта
товаров и остатков на складах для компании, занимающейся продажей игрушек.

## Технологии

- Python 3 + Tkinter (графический интерфейс)
- PostgreSQL (база данных, 3 нормальная форма, ссылочная целостность)
- psycopg 3 (подключение к PostgreSQL)
- Pillow (работа с изображениями товаров)

## Запуск

```
pip install -r requirements.txt
python3 main.py
```

Должен быть запущен сервер PostgreSQL. При первом запуске приложение само создаёт
базу данных `toystore` и наполняет её из скрипта `schema.sql`.

Параметры подключения можно переопределить переменными окружения:

| Переменная        | Назначение           | По умолчанию          |
|-------------------|----------------------|-----------------------|
| TOYSTORE_DB       | имя базы данных       | toystore              |
| TOYSTORE_USER     | пользователь          | текущий пользователь ОС |
| TOYSTORE_PASSWORD | пароль                | —                     |
| TOYSTORE_HOST     | хост                  | локальный сокет        |
| TOYSTORE_PORT     | порт                  | 5432                  |

Чтобы пересоздать базу с нуля: `dropdb toystore` и снова запустить приложение.

## Учётные записи

| Логин   | Пароль      | Роль          |
|---------|-------------|---------------|
| admin   | admin123    | Администратор |
| manager | manager123  | Менеджер      |
| client  | client123   | Клиент        |

Также со стартового окна можно войти как **гость** без авторизации.

## Права ролей

- **Гость / Клиент** — просмотр товаров без фильтрации, сортировки и поиска.
- **Менеджер** — просмотр товаров с фильтрацией, сортировкой, поиском; просмотр заказов.
- **Администратор** — всё перечисленное + добавление, редактирование и удаление товаров.

## Правила подсветки карточек

- Скидка больше 17% — фон `#FFDEAD`.
- Товар отсутствует на складе (количество 0) — фон голубой.
- При наличии скидки старая цена зачёркнута красным, рядом итоговая цена чёрным.

## Структура проекта

- `main.py` — точка входа, переключение окон, иконка приложения
- `database.py` — работа с PostgreSQL (подключение, запросы, авторизация, CRUD)
- `schema.sql` — скрипт создания и наполнения базы данных PostgreSQL
- `helpers.py` — ресурсы, изображения, расчёт итоговой цены
- `login_window.py` — окно входа
- `catalog_window.py` — каталог товаров (карточки, поиск, сортировка, фильтр)
- `product_form.py` — форма добавления/редактирования товара
- `orders_window.py` — окно просмотра заказов
- `resources/` — логотип и заглушка `picture.png`
- `product_images/` — изображения товаров, загруженные администратором
